# Compilador
El sistema analiza código fuente escrito en un lenguaje definido para el proyecto y lo transforma en una representación intermedia o código ejecutable mediante diferentes etapas del proceso de compilación.

El proyecto implementa componentes esenciales como análisis léxico, análisis sintáctico y validación semántica, permitiendo detectar errores en el código fuente y generar una estructura organizada del programa para su posterior procesamiento.
